# 🏦 CIBC Banking Analytics Dashboard

## 📌 Project Overview
An end-to-end data analytics project modeled after **CIBC (Canadian Imperial Bank of Commerce)**.  
This project covers the full data analytics lifecycle — from raw data ingestion to an interactive Power BI dashboard.

## 🎯 Objective
Analyze customer churn, segmentation, and financial KPIs using real-world banking data.
